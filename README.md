# IAM-users-listing-script
This script lists all the IAM-users in HashedIn account.<br/>

1. create a file<br/>

        vi IAM_listing.sh
   copy,paste the script and save it.<br/>
   
2. Run the script.<br/>

        sh IAM_listing.sh
